La **constante d'acidité** $K_A$ caractérise la force d'un couple [[acide-base]].

## Définition

Pour le couple $\text{AH}/\text{A}^-$, la réaction avec l'eau :
$$\text{AH} + \text{H}_2\text{O} \rightleftharpoons \text{A}^- + \text{H}_3\text{O}^+$$

$$K_A = \frac{[\text{A}^-][\text{H}_3\text{O}^+]}{[\text{AH}]}$$

## pKa

$$pK_A = -\log K_A$$

Plus $pK_A$ est **faible**, plus l'acide est **fort**.

## Constante de basicité

$$K_B = \frac{[\text{AH}][\text{HO}^-]}{[\text{A}^-]}$$

### Relation

$$K_A \times K_B = K_e = 10^{-14}$$
$$pK_A + pK_B = 14$$

## Échelle de pKa

| Couple | $pK_A$ |
|--------|--------|
| $\text{H}_3\text{O}^+/\text{H}_2\text{O}$ | 0 |
| $\text{H}_3\text{PO}_4/\text{H}_2\text{PO}_4^-$ | 2.1 |
| $\text{CH}_3\text{COOH}/\text{CH}_3\text{COO}^-$ | 4.8 |
| $\text{H}_2\text{CO}_3/\text{HCO}_3^-$ | 6.4 |
| $\text{NH}_4^+/\text{NH}_3$ | 9.2 |
| $\text{HCO}_3^-/\text{CO}_3^{2-}$ | 10.3 |
| $\text{H}_2\text{O}/\text{HO}^-$ | 14 |

## Diagramme de prédominance

À partir de [[Henderson-Hasselbalch]] :
- Si $pH < pK_A$ : forme **acide** ($\text{AH}$) prédomine
- Si $pH = pK_A$ : $[\text{AH}] = [\text{A}^-]$
- Si $pH > pK_A$ : forme **basique** ($\text{A}^-$) prédomine

>[!example] Acide acétique ($pK_A = 4.8$)
> - À $pH = 3$ : l'acide $\text{CH}_3\text{COOH}$ prédomine
> - À $pH = 7$ : l'ion acétate $\text{CH}_3\text{COO}^-$ prédomine
