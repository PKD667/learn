L'**état standard de référence** (ESR) d'un élément est son état le plus stable dans les conditions standard.

## Conditions standard

- Pression : $P° = 1$ bar
- Température : généralement $T = 298$ K ($25°C$)

## ESR des éléments courants

| Élément | ESR |
|---------|-----|
| Hydrogène | $\text{H}_2(g)$ |
| Oxygène | $\text{O}_2(g)$ |
| Azote | $\text{N}_2(g)$ |
| Carbone | $\text{C}$ graphite |
| Soufre | $\text{S}_8$ rhombique |
| Phosphore | $\text{P}_4$ blanc |
| Brome | $\text{Br}_2(l)$ |
| Mercure | $\text{Hg}(l)$ |
| Métaux | solide cristallin |

## Importance

Par convention, l'[[enthalpie de formation]] d'un élément dans son ESR est nulle :
$$\Delta_f H°(\text{élément dans son ESR}) = 0$$

Cela sert de **référence** pour toutes les grandeurs thermodynamiques.

>[!example] Carbone
> Le graphite est l'ESR du carbone (pas le diamant).
> - $\Delta_f H°(\text{C graphite}) = 0$
> - $\Delta_f H°(\text{C diamant}) = +1.9$ kJ·mol⁻¹
