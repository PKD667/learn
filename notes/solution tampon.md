Une **solution tampon** est une solution dont le [[pH]] varie très peu lors de l'ajout d'acide ou de base.

## Composition

Mélange de concentrations comparables de :
- Un **acide faible** $\text{AH}$
- Sa **base conjuguée** $\text{A}^-$

(ou une base faible et son acide conjugué)

## Pouvoir tampon

$$\beta = \frac{dC_\text{base ajoutée}}{dpH} = -\frac{dC_\text{acide ajouté}}{dpH}$$

Plus $\beta$ est grand, plus le tampon est **efficace**.

## Conditions d'efficacité

Le pouvoir tampon est maximal quand :
- $[\text{AH}] \approx [\text{A}^-]$
- $pH \approx pK_A$

Zone tampon : $pK_A - 1 < pH < pK_A + 1$

## Mécanisme

### Ajout d'acide ($\text{H}^+$)
$$\text{A}^- + \text{H}^+ \rightarrow \text{AH}$$
La base conjuguée neutralise l'acide ajouté.

### Ajout de base ($\text{HO}^-$)
$$\text{AH} + \text{HO}^- \rightarrow \text{A}^- + \text{H}_2\text{O}$$
L'acide neutralise la base ajoutée.

## Tampons biologiques

| Tampon | $pK_A$ | Milieu |
|--------|--------|--------|
| $\text{H}_2\text{CO}_3/\text{HCO}_3^-$ | 6.4 | Sang |
| $\text{H}_2\text{PO}_4^-/\text{HPO}_4^{2-}$ | 7.2 | Cellules |
| Histidine | 6.0 | Protéines |

>[!example] Tampon sanguin
> Le sang est tamponné à $pH = 7.4 \pm 0.05$ principalement par le système $\text{CO}_2/\text{HCO}_3^-$.
> 
> **Acidose** : $pH < 7.35$
> **Alcalose** : $pH > 7.45$
> 
> Les deux sont dangereux pour l'organisme.

>[!example] Préparation d'un tampon
> Pour préparer un tampon à $pH = 5.0$ avec l'acide acétique ($pK_A = 4.8$) :
> 
> $$5.0 = 4.8 + \log\frac{[\text{A}^-]}{[\text{AH}]}$$
> $$\frac{[\text{A}^-]}{[\text{AH}]} = 10^{0.2} \approx 1.6$$
