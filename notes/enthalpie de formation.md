L'**enthalpie de formation** $\Delta_f H°$ est l'[[enthalpie de réaction]] pour la formation d'une mole de composé à partir de ses éléments dans leur [[état standard de référence]].

## Définition

Pour un composé $\text{A}_a\text{B}_b$ :
$$a~\text{A(ESR)} + b~\text{B(ESR)} \rightarrow \text{A}_a\text{B}_b$$
$$\Delta_f H°(\text{A}_a\text{B}_b)$$

## Convention fondamentale

$$\Delta_f H°(\text{élément dans son ESR}) = 0$$

## Valeurs tabulées (25°C)

| Composé | $\Delta_f H°$ (kJ·mol⁻¹) |
|---------|-------------------------|
| $\text{H}_2\text{O}(l)$ | -286 |
| $\text{H}_2\text{O}(g)$ | -242 |
| $\text{CO}_2(g)$ | -394 |
| $\text{CO}(g)$ | -111 |
| $\text{NH}_3(g)$ | -46 |
| $\text{CH}_4(g)$ | -75 |
| $\text{C}_2\text{H}_6(g)$ | -85 |
| $\text{C}_6\text{H}_{12}\text{O}_6(s)$ | -1273 |
| $\text{NaCl}(s)$ | -411 |

## Utilisation

Via la [[loi de Hess]] :
$$\Delta_r H° = \sum_i \nu_i \Delta_f H_i°$$

>[!example] Formation de l'eau
> $\text{H}_2(g) + \frac{1}{2}\text{O}_2(g) \rightarrow \text{H}_2\text{O}(l)$
> 
> $\Delta_f H°(\text{H}_2\text{O}(l)) = -286$ kJ·mol⁻¹
> 
> Réaction très exothermique.
