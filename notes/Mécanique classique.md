
# Projections


#### Position

$$\vec{OM} = r\vec{u}_r$$

#### Vitesse

$$\vec{v} = \dot{r}\vec{u}_r + r\dot{\theta}\vec{u}_\theta$$

#### Accélération

$$\vec{a} = (\ddot{r} - r\dot{\theta}^2)\vec{u}_r + (r\ddot{\theta} + 2\dot{r}\dot{\theta})\vec{u}_\theta$$
#### Mouvement circulaire ($r = a = \text{cste}$ )

$$\vec{v} = a\dot{\theta}\vec{u}_\theta$$

$$\vec{a} = -a\dot{\theta}^2\vec{u}_r + a\ddot{\theta}\vec{u}_\theta$$
#### Dérivées des vecteurs unitaires

$$\frac{d\vec{u}_r}{dt} = \dot{\theta}\vec{u}_\theta$$

$$\frac{d\vec{u}_\theta}{dt} = -\dot{\theta}\vec{u}_r$$
