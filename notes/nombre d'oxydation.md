Le **nombre d'oxydation** (n.o.) est un nombre entier qui reflète l'état d'oxydation d'un élément dans une espèce chimique.

## Notation

En **chiffres romains** : $+I$, $+II$, $-I$, $0$, etc.

## Règles de calcul

1. **Corps simple** : n.o. $= 0$
2. **Ion monoatomique** : n.o. $=$ charge
3. **Hydrogène** : n.o.$(H) = +I$ (sauf hydrures : $-I$)
4. **Oxygène** : n.o.$(O) = -II$ (sauf peroxydes : $-I$)
5. **Somme** : $\sum$ n.o. $=$ charge de l'espèce

## Exemples

| Espèce | Élément | n.o. |
|--------|---------|------|
| $\text{Fe}$ | Fe | $0$ |
| $\text{Fe}^{2+}$ | Fe | $+II$ |
| $\text{Fe}^{3+}$ | Fe | $+III$ |
| $\text{H}_2\text{O}$ | H, O | $+I$, $-II$ |
| $\text{MnO}_4^-$ | Mn | $+VII$ |
| $\text{Cr}_2\text{O}_7^{2-}$ | Cr | $+VI$ |
| $\text{SO}_4^{2-}$ | S | $+VI$ |
| $\text{NH}_3$ | N | $-III$ |
| $\text{NO}_3^-$ | N | $+V$ |

## Interprétation

- **Oxydation** : le n.o. **augmente** (perte d'électrons)
- **Réduction** : le n.o. **diminue** (gain d'électrons)

## Utilité

Permet de :
1. Identifier les couples [[oxydoréduction|redox]]
2. Équilibrer les équations redox
3. Prévoir les réactions possibles

>[!example] Dismutation du chlore
> $\text{Cl}_2 + 2\text{HO}^- \rightarrow \text{Cl}^- + \text{ClO}^- + \text{H}_2\text{O}$
> 
> - $\text{Cl}_2$ : n.o. $= 0$
> - $\text{Cl}^-$ : n.o. $= -I$ (réduction)
> - $\text{ClO}^-$ : n.o. $= +I$ (oxydation)
> 
> Le chlore se dismute.
